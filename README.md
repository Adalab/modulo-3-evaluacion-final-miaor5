# Módulo 3: Ejercicio de evaluación final

## Descripción del proyecto

El objetivo de este proyecto es la creación de una página web con la que el usuario pueda buscar los mejores 'wow' del actor Owen Wilson. La pagina cuenta con dos filtros (nombre de la pelicula y año) para facilitar la busqueda del momento en el cuál el actor dijo su plabra celebre.

##### Puedes ver el proyecto entero en: http://beta.adalab.es/modulo-2-evaluacion-final-miaor5/

## Guía para arrancar el proyecto

> **NOTA:** Necesitas tener instalado [Node JS](https://nodejs.org/) para trabajar con este Starter Kit.

**Pasos a seguir :**

1. _Clonar este repositorio_
2. _Abrir una terminal en la carpeta raíz del repositorio_
3. _Instalar las dependencias locales ejecutando en la terminal el comando:_

```bash
npm install
```

**Arrancar el proyecto con el comando**

```bash
npm start
```
